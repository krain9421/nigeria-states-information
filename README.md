# nigeria-states-information API

## Description
The NaijaStates API is a RESTful service that provides detailed information about Nigerian states, including their names, capitals, populations, area sizes, and more.

## Technologies Used
- Node.js
- Express.js
- MongoDB
- NGINX
- Mongoose
- dotenv
